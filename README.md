# Mobile-Sales-Power-BI-Dashboard and Data Analysis

Project Overview:

This project analyzes sales data to uncover business insights and visualize key performance indicators (KPIs) using SQL and Power BI. The dashboard helps stakeholders monitor sales performance, customer trends, product performance, and regional revenue distribution.



<img width="425" height="239" alt="image" src="https://github.com/user-attachments/assets/322f2617-f0f3-43d9-8714-07a33fb8c116" />

Objectives:

Analyze sales performance across different countries and states.
Identify top customers and best-selling product lines.
Track monthly revenue trends.
Measure order fulfillment status.
Generate actionable business insights through interactive dashboards.
Tools & Technologies
SQL (PostgreSQL) – Data Analysis.
Power BI – Interactive dashboard creation and visualization.
Excel/CSV – Data source.
